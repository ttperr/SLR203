# SLR203

>My repository for the SLR 203 MQTT exercises with PAHO at Télécom Paris.

## Installation

### Prerequisites

* Java 8
* MQTT - Spy Client (Paho GUI) (recommended)
* JoramMQ Server

### Instruction

1. Clone the repository
2. Lunch JoramMQ Server
3. (Lunch MQTT-Spy Client to spy messages)
4. Enjoy
